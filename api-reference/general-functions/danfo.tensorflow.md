# danfo.tensorflow

danfo.**tensorflow**

**Returns:**

> &#x20;return [Tensorflow.js](https://www.npmjs.com/package/@tensorflow/tfjs) library

{% tab title="Node" %}

```javascript
const dfd = require("danfojs-node");
const tf = dfd.tensorflow;

let tensor_arr = tf.tensor2d([
  [12, 34, 2.2, 2],
  [30, 30, 2.1, 7],
]);
console.log(tensor_arr);
```
